# Personal Website

This repo contains all the code and posts related with my personal website

## How to run?

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd samuelsilvadev.github.io
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run develop
```
